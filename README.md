# MeFirstProject
A personal web project created to document memories at Teofilo V. Fernandez National High School. Built using pure HTML and CSS.


### Teofilo Memories / Aguijournal
This is my first web development project, created as a performance task for Grade 10 at Teofilo V. Fernandez National High School.

### The Goal
To create a multimedia journal that preserves memories of my junior high school years while practicing HTML and CSS fundamentals.

### Features
• esponsive layout for mobile and desktop.
• Imaage gallery of school milestones.
• Custom CSS styling (Glassmorphism/Aesthetic theme).

### What I Learned
• How to structure a multi-section webpage using HTML5.
• How to use CSS Flexbox to align elements.
• The process of deploying a site live using GitHub Pages.
